How to Read Codes

Capacitor codes typically use a 3-digit system followed by a letter:

  • 1st & 2nd Digit: Significant figures (Base value).
  • 3rd Digit: Multiplier (Number of zeros to add).
  • Letter: Tolerance (e.g., J=±5%, K=±10%, M=±20%).
  • Base Unit: The result is always in Pico-farads (pF).

Pro Tips

  • 104 is the most common bypass capacitor (0.1 µF).
  • Voltage Rating: Often marked separately (e.g., 1H = 50V, 2A = 100V). If not marked, assume low voltage (50V).
  • Ceramic vs Electrolytic: These codes are mostly for ceramic/film capacitors. Electrolytics usually write the value directly (e.g., 10µF).
