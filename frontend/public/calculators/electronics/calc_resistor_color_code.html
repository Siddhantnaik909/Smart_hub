4-Band Resistor Decoder

Translate colored bands into precise resistance values.

How It Works

Standard 4-band resistors use a color code to indicate value and tolerance:

  • Band 1 & 2: Significant digits (e.g., Brown=1, Black=0 → 10).
  • Band 3: Multiplier (Power of 10).
  • Band 4: Tolerance (Gold=±5%, Silver=±10%).

Pro Tips

  • Orientation: The tolerance band (usually Gold or Silver) is typically separated from the others. Hold this band to the right.
  • Mnemonics: "Big Boys Race Our Young Girls But Violet Generally Wins" (Black, Brown, Red, Orange, Yellow, Green, Blue, Violet, Grey, White).
