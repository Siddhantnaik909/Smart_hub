How It Works & Tips

Calculation Logic:
  • The duration is calculated based on calendar years, months, and days.
  • When excluding weekends/holidays, the tool iterates day-by-day between the start and end dates to count only the included days.
  • The doughnut chart visualizes the proportion of working days vs. non-working days in the selected period.
Pro Tips:
  • Use "Include End Date" for calculations where both start and end days count (e.g., hotel stays).
  • You can paste a list of dates (YYYY-MM-DD) separated by commas, spaces, or new lines into the holidays box.

How It Works & Tips

Calculation Logic:

This tool calculates a target date by adding or subtracting a specified number of calendar days from today's date. It does not exclude weekends or holidays.

Pro Tips:
  • Enter a positive number (e.g., 90) to find a date in the future.
  • Enter a negative number (e.g., -30) to find a date in the past.

How It Works & Tips

Calculation Logic:

The calculator starts from your specified date and counts forward one day at a time. It only increments the "business days" counter if the current day is not a Saturday, Sunday, or a date listed in the Public Holidays list.

Pro Tips:
  • The holiday list is shared across all calculators on this page.
  • Click the "Skipped" day counts to see a detailed list of the exact dates that were excluded.

Calculation Logic:

This tool works in reverse of the "Add" calculator. It starts from the end date and counts backward, skipping weekends and holidays until the target number of business days has been subtracted.

Pro Tips:
  • This is useful for finding project start dates based on a deadline and a known number of workdays.
